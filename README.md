# Храм-часовня святителя Николая Чудотворца

Одностраничный мобильный сайт-открытка для размещения за QR-кодом. Содержит историю храма, фотографии, аудиорассказ и расписание богослужений.

## 📱 Особенности

- Адаптивный дизайн для мобильных устройств
- Слайдшоу с поддержкой свайпов (влево/вправо)
- Кастомный аудиоплеер
- Две информационные карточки (история и расписание)
- Оптимизирован для быстрой загрузки

## 🛠 Технологии

- HTML5
- CSS3 (Flexbox, Grid, анимации)
- JavaScript (ES6+)
  ![HTML5](https://img.shields.io/badge/HTML5-orange) ![CSS3](https://img.shields.io/badge/CSS3-blue) ![JavaScript](https://img.shields.io/badge/JavaScript-yellow)

## 🚀 Установка и запуск

1. Скачайте все файлы проекта
2. Добавьте свои фотографии в папку `images/`
3. Добавьте аудиорассказ в папку `audio/`
4. Откройте `index.html` в браузере

## ⚙️ Настройка

### Фотографии

- Поддерживаются форматы JPG, PNG
- Рекомендуемый размер: 800×600 пикселей
- Имена файлов: только латиница, без пробелов

### Аудио

- Формат: MP3 (рекомендуется 64-96 kbps)
- Длительность: любая
- Размер: желательно до 5 МБ
